# The_perfect_guess
<br>
A simple Python console-based Number Guessing Game where the computer generates a random number between 1 and 100, and the player tries to guess it.

## 🚀 Features

- Random number generation using Python's `random` module
- User-friendly hints:
  - "Higher Number please"
  - "Lower number please"
- Tracks the number of attempts taken
- Simple command-line interface

## 🛠️ Technologies Used

- Python 3
- Random Module

## 📌 How It Works

1. The program generates a random number between 1 and 100.
2. The user enters a guess.
3. The program provides hints:
   - If the guess is too high, it asks for a lower number.
   - If the guess is too low, it asks for a higher number.
4. The game continues until the correct number is guessed.
5. The total number of guesses is displayed.
